## Content updates via JSON

The interactive career timeline now loads its content from `data/career-phases.json`.

- Update role wording in the `role` field.
- Update narrative wording in the `story` field.
- Update technology wording in the `tech` field.
- Update challenge wording in the `challenges` field.
- Update chart levels in the `skills` array.

### `skills` array order

Use this order for each role:

1. Infrastructure & Systems
2. Product Engineering (App Dev & Product Mindset)
3. Cybersecurity
4. Enterprise Architecture
5. Strategic Leadership
6. FinOps

After editing JSON, refresh the page to see changes reflected in the interactive charts and role cards.
